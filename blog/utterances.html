Introduction

Today I have accomplished something (with the help of a lil AI...) revolutionary. My blog posts have comments. But how could I accomplish such a thing in the age of corporate garbage infecting the popular solutions like Disqus? Well, with the help of Utterances.

But what is Utterances?

Utterances is a comment system built upon GitHub issues. This means that you can keep your comments with your repository (or in a seperate repository). This also means that you don't have to run any server code, and you're not giving some random company all your data (except for GitHub but my site is already on GitHub so I don't really care).

The only problem(s)

However, Utterances isn't perfect (as is nothing) and it has a couple problems. Namely of which is that you cannot reply to comments, which is quite irritating, but you can just @mention people so it's "fine".

The more big problem is that it doesn't support (easy) custom styling right of the box. And this is because, and I quote DuckDuckGo Search assist, "due to same-origin policy restrictions". Whatever the hell that means... So, I can't have cool matching styling...

I might switch to Giscus, but that seems hard 😔
